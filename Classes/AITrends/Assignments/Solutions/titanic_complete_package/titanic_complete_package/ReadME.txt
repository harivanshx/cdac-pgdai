Titanic ML Assignment - Complete End-to-End (Student-friendly)

1️Download Titanic data from Kaggle:
   https://www.kaggle.com/competitions/titanic/data
   Place the `train.csv` file inside the `data/` folder.

2️Install requirements:
   pip install -r requirements.txt

3️Train all models:
   python src/train.py

4️Evaluate a model:
   python src/test.py --model models/model_logistic.pkl

5️Run the Streamlit app:
   streamlit run src/app.py
